#### **TOKEN**

\->Token is a small part of the program

\->There are five different kind of Tokens : Identifiers (x), Literals (10), Keywords (if), Operators(+), Punctuators(:)



#### **STATEMENT**

\->Statement is a single line of a program that performs some actions



#### **COMMENTS**

\->Comments are used to read the code for better understanding

\->There are two type of comments:->Single Comments

&#x20;                                ->Multiple Line Comments

\->Single line comments are represented by #

\->Multiple line comments are represented by '''   MULTIPLE LINES  '''



#### **VARIABLE RULES**

\->Variable can be defined by using uppercase(A-Z),lowercase(a-z),numbers(0-9),\_

\->Variable name can only start with (A-Z),(a-z),\_

\->No keyword are used as variables

\->These are Case sensitive



**VARIABLE ASSIGNMENT**

\->If we need to assign a single value to every variable then use a=b=c=10

\->If we need to assign a different values to different variables in a single line then mention=a, b, c=10,20,30

\->If we need to swap values of a variable then a, b= b, a
